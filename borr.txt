<div class="prose dark:prose-invert max-w-none">
  <p>
    Great! Here‚Äôs a concise summary of the key points for the
    <strong>0/1 Knapsack Problem</strong> followed by the Python
    implementation.
  </p>
  <h3>Key Points Summary:</h3>
  <ul>
    <li>
      <p>
        <strong>DP Table Definition</strong>: Create a 2D array
        <div
          style="color: rgb(212, 212, 212); font-size: 13px; text-shadow: none; font-family: Menlo, Monaco, Consolas, 'Andale Mono', 'Ubuntu Mono', 'Courier New', monospace; direction: ltr; text-align: left; white-space: pre; word-spacing: normal; word-break: normal; line-height: 1.5; tab-size: 4; hyphens: none; padding: 1em; margin: 0.5em 0px; overflow: auto; background: rgb(30, 30, 30);"
        >
          <code
            class="language-text"
            style="white-space: pre; color: rgb(212, 212, 212); font-size: 13px; text-shadow: none; font-family: Menlo, Monaco, Consolas, 'Andale Mono', 'Ubuntu Mono', 'Courier New', monospace; direction: ltr; text-align: left; word-spacing: normal; word-break: normal; line-height: 1.5; tab-size: 4; hyphens: none;"
            ><span>dp</span></code
          >
        </div>
        where
        <div
          style="color: rgb(212, 212, 212); font-size: 13px; text-shadow: none; font-family: Menlo, Monaco, Consolas, 'Andale Mono', 'Ubuntu Mono', 'Courier New', monospace; direction: ltr; text-align: left; white-space: pre; word-spacing: normal; word-break: normal; line-height: 1.5; tab-size: 4; hyphens: none; padding: 1em; margin: 0.5em 0px; overflow: auto; background: rgb(30, 30, 30);"
        >
          <code
            class="language-text"
            style="white-space: pre; color: rgb(212, 212, 212); font-size: 13px; text-shadow: none; font-family: Menlo, Monaco, Consolas, 'Andale Mono', 'Ubuntu Mono', 'Courier New', monospace; direction: ltr; text-align: left; word-spacing: normal; word-break: normal; line-height: 1.5; tab-size: 4; hyphens: none;"
            ><span>dp[i][j]</span></code
          >
        </div>
        represents the maximum value achievable with the first ( i ) items and a
        maximum weight of ( j ).
      </p>
    </li>
    <li>
      <p><strong>Base Case</strong>:</p>
      <ul>
        <li>
          If there are no items or the capacity is zero: [ dp[0][j] = 0 \quad
          \text{for all } j ] [ dp[i][0] = 0 \quad \text{for all } i ]
        </li>
      </ul>
    </li>
    <li>
      <p><strong>Recurrence Relation</strong>:</p>
      <ul>
        <li>
          If the weight of the current item ( w_i ) is less than or equal to the
          current capacity ( j ): [ dp[i][j] = \max(dp[i-1][j], dp[i-1][j - w_i]
          + v_i) ]
        </li>
        <li>
          If the weight of the current item is greater than the current
          capacity: [ dp[i][j] = dp[i-1][j] ]
        </li>
      </ul>
    </li>
    <li>
      <p>

          <p><strong>Time Complexity</strong>:</p>
          <ul>
            <li>
              The most time-consuming step in the Fractional Knapsack algorithm
              is sorting the items based on their value-to-weight ratio. This
              sorting step has a time complexity of \( O(n \log n) \), where \(
              n \) is the number of items.
            </li>
            <li>
              After sorting, the algorithm iterates through the items to fill
              the knapsack, which takes \( O(n) \) time in the worst case.
            </li>
            <li>
              Therefore, the overall time complexity of the Fractional Knapsack
              Problem is: \[ O(n \log n) + O(n) = O(n \log n) \]
            </li>
          </ul>
        </li>
        <li>
          <p><strong>Space Complexity</strong>:</p>
          <ul>
            <li>
              The space complexity is primarily determined by the storage of the

              complexity is \( O(n) \).
            </li>
            <li>
              If we only use a few additional variables for tracking the total
              value and remaining capacity, the additional space used is
              constant, \( O(1) \).
            </li>
            <li>Thus, the overall space complexity is: \[ O(n) \]</li>
          </ul>
        </li>
